The Challenge: To build a high-performance information retrieval system capable of processing boolean queries (AND/OR) and ranking documents by relevance, moving beyond simple linear search.

The Solution:
Optimization: Implemented Inverted Indexes using Binary Search Trees (BST) to drastically reduce search complexity from O(n) to O(log n).
Search Logic: Developed a query parser to handle complex Boolean operations (AND, OR, NOT) and Ranked Queries based on term frequency.
Data Handling: Utilized custom Linked Lists and BSTs to manage document metadata and stop-word filtering efficiently.
Technologies: Java, Binary Search Trees (BST), Big-O Optimization, Inverted Indexes.
